# Simple Task Manager  
This project is designed to help users keep track of their tasks and improve productivity by providing a clean and simple interface.  

## Features
Add new tasks  
Edit existing tasks  
Delete tasks  
View and manage all tasks in one place  
Simple and user-friendly interface
## Technologies Used
JavaScript  
HTML  
CSS
